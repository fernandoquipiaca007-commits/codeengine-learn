🔍 PROBLEMAS PROVÁVEIS
═══════════════════════════════════════════════════════════════════════════

🟡 Problema #1: Webhooks não funcionando (70% probabilidade)
   → Webhooks não estão chegando ao backend
   → Compras não são registradas automaticamente
   → SOLUÇÃO: Configurar Stripe CLI + webhook endpoint

🟡 Problema #2: Metadata incompleto (20% probabilidade)
   → create-checkout não envia member_id
   → fulfill-purchase não consegue processar
   → SOLUÇÃO: Corrigir metadata + reprocessar compras

🟡 Problema #3: Usuários sem members (10% probabilidade)
   → Trigger de criação não existe ou falhou
   → Usuários fazem login mas não têm registro em members
   → SOLUÇÃO: Criar trigger + migrar usuários existentes
