# Exemplo de Documentação - Store/Frontend

## Visão Geral
Este é um arquivo de exemplo para a estrutura de documentação do módulo Store (Frontend).

## Componentes Principais
- Pages
- Components
- Layouts
- Hooks

## Instalação
```bash
npm install
npm run dev
```

## Estrutura de Pastas
```
/src
  /pages
  /components
  /hooks
  /utils
  /styles
```

## Convenções
- Use React + TypeScript
- Mantenha componentes reutilizáveis
- Siga o padrão BEM para CSS
- Componentes nomeados com PascalCase
